# 📖 Dictionary Quick Start

## 🚀 3-Second Tutorial

1. **Click the floating book icon** 📖 (bottom-right corner)
2. **Type a word** in Japanese or English
3. **Press Enter** or click search 🔍

That's it! You're now looking up words like a pro! 🎉

## 🎮 During Gameplay

### Scenario: You see an unknown word

**🤔 Opponent plays:** `桜` (What is this?!)

**👆 You:**

1. Click dictionary button (game continues!)
2. Search "桜" or "sakura"
3. See: **Cherry blossom** 🌸 (N5, Common word)
4. Click 🔊 to hear: "さくら"
5. Close dictionary
6. Play your turn with newfound knowledge!

**🎓 Result:** You learned a new word without pausing the game!

## 🔊 Audio Pronunciation

### How to Use

1. Search for any word
2. Click the **🔊 speaker icon** next to the reading
3. Listen to native Japanese pronunciation
4. Repeat 3 times for better retention!

### Best Results

- ✅ Use Chrome or Edge browser
- ✅ Enable audio permissions
- ✅ Use headphones for clarity

## 🏷️ Understanding Tags

When you search a word, you'll see colored tags:

### 🟢 **Common**

- Word is frequently used
- Great for beginners to learn
- Safe to use in conversation

### 🟡 **JLPT N5-N1**

- **N5** → Beginner (easiest)
- **N4** → Elementary
- **N3** → Intermediate
- **N2** → Pre-advanced
- **N1** → Advanced (hardest)

### 🔵 **Parts of Speech**

- **Noun** - Thing/person/place
- **Verb** - Action word
- **Adjective** - Describing word
- **Adverb** - Modifies verb

## 🎨 Dictionary Modes

### 📦 **Minimized Mode**

- Small circle with 辞書 (dictionary)
- Doesn't block game view
- Click to restore

### 📄 **Normal Mode**

- 400px width
- Shows top result + 5 alternatives
- Perfect for quick lookups

### 📰 **Expanded Mode**

- 600px width
- Shows top result + 10 alternatives
- More definitions per word
- Click **⬜** icon to expand

## 🎯 Pro Tips

### Fastest Search Methods

**Method 1: Copy-Paste**

```
1. Copy unknown word from chat/game
2. Click dictionary
3. Paste and press Enter
4. Instant result!
```

**Method 2: Type Romaji**

```
Instead of: いぬ
Type: inu
Works the same!
```

**Method 3: English → Japanese**

```
Search: "dog"
Get: 犬 (いぬ)
```
